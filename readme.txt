== Description ==

The plugin widget Specific Posts. With this plugin you can put a post on widget. Works with Elementor.

== Installation ==

= Instalação =
1. Upload to the /wp-content/plugins/directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to the Appearance and Widget menu.
4. Add the Specific Posts Widget and enter the post number.
